
# calendar

## ✅ 프로젝트 소개
  바닐라 자바스크립트로 구현한 Calendar & DatePicker
<br /><br />

## 🧑‍🤝‍🧑 멤버 구성
  개인 프로젝트입니다.
<br /><br />

## 🔗 배포 링크
  https://calendar-beta-seven.vercel.app/
<br /><br />

## ⚙ 개발 환경
  - `JavaScript`
<br /><br />

  - calendar를 모듈화하여 재사용이 가능하도록 구현
